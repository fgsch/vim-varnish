# vim-varnish

Vim and Neovim syntax highlighting support for Varnish Configuration
Language (VCL).

## Installation

### Using vim-plug

1. Add `Plug 'fgsch/vim-varnish'` to `~/.vimrc`
2. `:PlugInstall` or `$ vim +PlugInstall +qall`
